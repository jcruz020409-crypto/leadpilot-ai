# Roadmap

Status: draft

## Phase 1: Demo 1

- URL intake.
- Manager Agent orchestration.
- Research Agent website and allowed public context analysis.
- Opportunity Agent Detected Problems.
- Pricing Agent Sales Opportunity Score and Estimated Budget.
- Proposal Agent Proposal Draft, email, and work plan.

- Lead list intake.
- CRM-style opportunity board.
- Approved outbound email.
- Reply classification.
- Follow-up recommendations.

## Phase 4: Integrations

- CRM sync adapters.
- Email provider adapters.
- Enrichment provider adapters.
- Team workspace and collaboration controls.
