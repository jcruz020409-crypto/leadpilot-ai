# Prompts Library

Status: draft

## Prompt Families

Prompt templates must be versioned and tied to `agent_runs`.

## Manager Agent

Purpose: create the analysis plan, sequence agents, verify output completeness,
and assemble the Final Analysis Report.

## Research Agent

Purpose: summarize the company from website content, allowed public social
signals, and sector context. It must flag missing or uncertain information.

## Opportunity Agent

Purpose: detect problems, possible automations, and improvements with evidence,
business impact, and confidence.

## Pricing Agent

Purpose: calculate complexity, ROI assumptions, Sales Opportunity Score, and
Estimated Budget.

## Proposal Agent

Purpose: generate Proposal Draft, outreach email, plan of work, investment
summary, and Recommended Next Steps.

## Memory Agent

Purpose: save analyzed company summaries, prior decisions, and user preferences
without storing secrets or unnecessary personal data.
